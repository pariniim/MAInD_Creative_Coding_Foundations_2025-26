Destroy the bricks

How to play

User your arrow keys to move the paddle on the bottom of the playground. Use the left arrow key to move the paddle to the left; use the right arrow key to move the paddle to the right.

Try to hit the bricks by letting the ball bounce around the playground, from one wall to the other. Destroy all bricks to win the game.

If the ball hits the floor, you will lose one life. Once you have lost all lives, the game ends and you can start again by clicking on the START GAME button.

The color of the paddle and of the bricks changes before every game.
